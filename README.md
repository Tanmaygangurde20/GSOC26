

🚀 GSOC 2026 – Quantum Circuit Design with LLMs

This repository contains my Google Summer of Code 2026 proposal and work for the organization ML4SCI (CERN).

📌 Project Overview

The project focuses on building an autonomous agent framework powered by Large Language Models (LLMs) to design, validate, and optimize quantum circuits for Variational Quantum Eigensolver (VQE) applications.

Instead of manual trial-and-error, the system introduces a closed-loop intelligent workflow:

Generate quantum circuits using an LLM agent

Validate circuits before simulation (syntax, semantics, topology)

Execute valid circuits on a quantum simulator

Compute physics-aware metrics (expressibility, entanglement, depth)

Iteratively improve circuit design using structured feedback
